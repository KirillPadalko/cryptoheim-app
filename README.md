# CryptoHeim Landing Page

A modern, premium landing page for the CryptoHeim mobile app.

## Project Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at the URL shown (usually `http://localhost:5173`).

## Features

- **Responsive Design**: Adapts to mobile and desktop.
- **Modern Animations**: CSS transforms and transitions.
- **Premium Aesthetics**: Gradient branding and glassmorphism.
